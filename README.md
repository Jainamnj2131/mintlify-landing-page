# Mintlify Hero Section Clone

## Sections Implemented

- Transparent Navigation Bar
- Hero Section with Background Image
- Gradient Headline
- Email Input + CTA Button
- Dashboard UI Preview

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter)

## Color & Styling

- Dark background (#000000)
- White primary text
- Soft gray secondary text (#d1d5db)
- Green gradient highlight in heading
- Linear gradient overlay on hero background
